# Photonic Integrated Circuit (PIC) Portfolio

This repository presents a collection of Photonic Integrated Circuit (PIC) layouts and design projects completed during the PIC-UPV PERTE CHIP Specialized Training Program.

The objective of this repository is to demonstrate practical experience in:

PIC layout design
Silicon photonics
GDSFactory
Hierarchical cell design
Optical routing
Passive photonic devices
Design reproducibility from research publications
