# Photonic Integrated Circuit (PIC) Design 

This repository contains my Photonic Integrated Circuit (PIC) design projects developed during my research and training in integrated photonics. It includes the design, simulation, and layout generation of various passive photonic components using Python, GDSFactory, and Lumerical.

## Repository Contents

- Waveguide Design
- Mach–Zehnder Interferometer (MZI)
- Multimode Interference (MMI) Couplers
- Ring Resonators
- Directional Couplers
- GDS Layout Generation
- Simulation Results
- Python Scripts

## Software Used

- Python
- GDSFactory
- Lumerical MODE
- Lumerical FDTD
- Tidy3D (where applicable)

## Research Interests

This repository reflects my interest in Photonic Integrated Circuits, integrated optical sensors, waveguide design, and photonic device fabrication.
